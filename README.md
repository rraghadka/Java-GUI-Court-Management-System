# Java GUI Court Management System

This project is a desktop application developed in Java using Swing.
It simulates a Court Management System with a graphical user interface (GUI).

## Features

* Login system (Admin and users)
* User-friendly interface
* Organized project structure (MVC style)
* Data management and interaction

## Technologies

* Java
* Swing (GUI)

## How to Run

1. Open the project in Eclipse
2. Run the main class (LoginFrame.java)
3. Use the following credentials:

   * Username: ADMIN
   * Password: ADMIN

## Project Structure

* `control/` – handles logic
* `model/` – data and entities
* `view/` – GUI screens
* `enums/` – enumerations
